Components

Introduction

We leverage CSS to lay elements out on the page. However, there is a distinction between layouts dictating the major and minor components of a page. The minor are called Modules and include Accordions, Modals, and Tabs; while the major Components include the header and footer.

Examples

Bad CSS structure might look something like this:

<!-- examples of bad naming conventions -->
<!doctype html>
<?php
    $pageTitle = 'U.S. Energy Information Administration (EIA)';
    $L2Title = ''; // Contact Us
    $locale =  '' // about
?>
<html>
<head>
    <?php
    include('adaptive/global/head/includes/head.inc'); // required
    ?>
    <title>
        <?=$L2Title?> - <?=$pageTitle?> -   
    </title>   
</head>

<body>
    <?php
    include('adaptive/global/header/includes/header.inc'); // required
    ?>
    <!-- see the layouts section -->
    <?php include('adaptive/global/footer/includes/footer.inc') ?> // required
</body>
</html>

Good CSS structure might look something like this:

<!-- default row -->
<div class="l-row"></div>
