Layout Rules

Introduction

We leverage CSS to lay elements out on the page. However, there is a distinction between layouts dictating the major and minor components of a page. The minor are called Modules and include Accordions, Modals, and Tabs; while the major Components include the header and footer.

Naming Conventions

  • All style naming conventions use classes and not ID selectors.
  • All class names are lowercase and hyphenated.
  • Layout class names begin with a l-.

The Structure

Layouts are brokendown into two parts, Rows and Columns.

All content should live within a row in order for it to responsively adapt to the 15px16px16px12px 12px page wrapper.

There's only two types of row class names, l-row and l-row header. The difference it that the default row adds a 22px space on the left and sometimes has a 22px space on the right (based on the type of content). As a developer or content editor, always use the l-row as the other is reserved for special applications.
