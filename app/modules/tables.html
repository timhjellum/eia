• Tables

    Purpose

    Users need to see structured data in a manner that makes the data readable, scannable, and comparable.

    Description

    Tables provide customizable views of tabular data with features including sorting and managing columns. Tables are useful when a collection of data requires vertical alignment through columns for quick scanning and discovery of information, as well as for data that with many attributes that requires sorting, grouping, and a user customizable display.

    Basic functionality

    Tables list a collection of items in rows that have similar attributes, usually with headers to label each attribute.

  • Highlight Columns

    Highlighting a column (body and foot, not the head) by adding a highlight-col-x to the table class.

    Example:

    <!-- highlight the second and fourth columns -->
	<table class="highlight-col-2 highlight-col-4"></div>

  • Align Columns

    Overiding the column alignment (head, body and foot) can be accomplished by adding a align-left-col-x or align-right-col-x to the table class.

    Example:

    <!-- align left the second column, and center the third column (the fourth column is right aligned by default) -->
	<table class="basic-table full-width align-left-col-2 align-center-col-3"></table>
