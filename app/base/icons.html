Icons

The grapjic images below are SVG, an image format for vector graphics. These images are not physical images that you link to like a typical .gif, .jpg or .png. Vvector-based icons look great in a high pixel density world and offers design control, and predictability. The examples below are the base examples with minimal customizing. They also have a nested span and label which is used for accessibility.

  • External Link

    Using CSS rules in the base.less file. The rules will add an external link icon to all links that have a http://, https://, and .com in the URL except if eia.gov is in the file name. Additionally, special rules remove the external link from the .social (vertical slider) and .bx-viewport (sliders). To force an external link you can simply add a class="external-link" to the anchor tag.
