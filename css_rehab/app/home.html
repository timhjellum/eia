Favicon

The EDC favicon (short for favorite icon), also known as a shortcut icon, website icon, tab icon, URL icon or bookmark icon, is a file named favicon.ico that often displays in the browser tab, browser bookmarks and browser URL input field.

  • Favicon:

  • Description: Overides the default SharePoint icon with a EDC favicon.

  • Width: 16 pixels.

  • Height: 16 pixels.

  • Path: https://educationdevelopmentcenter.sharepoint.com/sites/intranet/Style%20Library/Images/

  • Filename: favicon.ico
