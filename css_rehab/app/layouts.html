Layouts

Introduction

Layout rules divide the page into sections. Layouts hold one or more modules together.

These layouts were created from examining hundreds of eia.gov pages. The new layouts simply take the existing fixed pixel width layouts and makes them percentages. So far we have identified x top-level layouts. These include:

After inventory of the site it was determined that there was little or no difference in the main-col and side-col widths.

  1. Full-Width-Header Layout (22px + 100% + 22px) or 956px
  2. Full-Width Layout (100%)
  3. Left & Right Column (25% / 50% / 25%) 228px + 456px + 228px = 912px
  4. Right-Column Layout (75% / 25%) 684px + 228px = 912px
  5. Left-Column Layout (25% / 75%) 228px + 684px = 912px
  6. Wide-Right-Column Layout (41.333% / 58.666%)
  7. Wide-Left-Column-Layout (58.666% / 41.333%)
  8. One-Third-Column-Left Layout (33.333% / 66.666%) 304px + 608px = 912px
  9. One-Third-Column-right Layout (66.666% / 33.333%) 608px + 304px = 912px
  10. Split Column Layout (50% / 50%) 456px + 456px = 912px
  11. Three Column Layout (33.333% / 33.333% / 33.333%) 304px + 304px + 304px = 912px
  12. Four Column Layout (25% / 25% / 25% / 25%) 228px + 228px + 228px + 228px = 912px

The problem is that these get really really close to an existing layout, but might be off by a few pixels. No biggie, we have a solution for this.

The most common Desktop and Laptop resolutions display a centered and 980 pixel fixed-width layout. The margin is 15px and is simulated with the following color: . Some content, usually page-headers and sliders, use this layout. To denote this the Double Class Selector of .row .header is used. the Desktop and Laptop resolution width is [990 pixels - 4 pixels (inner and outer styles)] - 30 pixels = 956 pixels.

All else has a 22 pixel simulated with the following color: .
