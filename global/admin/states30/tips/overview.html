Overview

The States 3.0 Admin Panel manages the faceted search tool on the "Find" page in the U.S. States section of the web site. The faceted search tool allows users to quickly find State level information from across the EIA website.

Content from the website to be included in this search tool must be defined or "tagged" for various characteristics. Broad classes of characteristics like the energy source, type of activity and area make up the facets. The most important facets are used as drop-down filters in the States 3.0 faceted search tool.

The Admin Panel serves several functions in support of the faceted search tool:

  • Tagging of content to be included in the search tool
  • Determining the order or "weighting" of results that are generated from the search tool
  • Defining and formatting the items that appear in the search tool drop-down filters.

While this system can be extended to all content on the web site, for the time being it is limited to State level data as related to States 3.0.

Tagging Content

Use the "Add a New CPR" page to tag content.

A Content Profile Record (CPR) is a set of tags (or major characteristics) that describe a piece of web content.

Content that should be tagged:

  • Individual data tables that make up a publication, table browser, or stand-alone page.
  • Articles that are stand-alone or parts of a publication.
  • Visualizations such as maps, charts, video, etc.
  • Presentations
  • Documentation (for models, systems)
  • Survey forms and instructions.

Content should not be tagged:

  • Publications as a whole. The components of publications are tagged (e.g., individual data tables, analysis, maps, charts, etc.). General information about publications is entered into a separate publication screen.
  • Old archived versions of recurring products.
  • Central (non-product specific) landing pages such as the EIA Home Page, L2 Pages.
  • Directory lists such as the pages that make up the data and analysis tabs and archived listings.
  • Background information such as source notes, explanatory notes, and technical details.
