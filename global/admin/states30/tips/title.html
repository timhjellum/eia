Title

Give the title as shown on the content page with the following possible adjustments:

  • Do not include table numbers or chapter numbers.
  • For recurring products, do not include a date.
  • For recurring historical data tables include a starting date but not the current date; Example: Natural Gas Well Activity, Beginning 1960.

For a dynamic link (see URL section below) with State level information, the string "[STATE_NAME]" can be used to dynamically generate a State name in the title.
