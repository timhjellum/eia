URL(s)

Add the URL for the primary landing page. Add a new URL for each alternative format (pdf, xls, csv, etc.).

Do not use a fully qualified URL. Begin the URL after www.eia.gov

Correct: /coal/index.cfm

Incorrect: http://www.eia.gov/coal/index.cfm

Type

Select the correct format type from the drop-down for each URL added. ColdFusion pages (.cfm) should be tagged as "HTML".

Dynamic

A set of State based URLs that differ only by a state code can be dynamically generated. In order to use this feature, you must replace the state code or state name within the URL with one of the following strings:

  • 2-character State codes should be replaced by "[STATE_CODE_2]".
  • State names with spaces (e.g. "North Dakota") should be replaced by "[STATE_NAME]".
  • A State name without spaces should be replaced by "[STATE_NAME_NOSPACE]".

The set of States to be dynamically generated is defined in the Area drop-down list.

To activate this feature set the Dynamic box to "Yes".
